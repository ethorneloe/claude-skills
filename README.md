# claude-skills

Custom skills for `Claude.ai` Each subdirectory is a self-contained skill package (uploadable as a `.zip`).

## beautiful-mermaid
Renders Mermaid diagrams via the [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) library with CSS custom properties, Light/Dark theme switching, zoom/pan controls, and full-viewport viewing.
